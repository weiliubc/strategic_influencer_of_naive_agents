Time-variant dynamic system¶

Using the paper as a basis, let:

$$V_t(x_t) = \max_{r_t} -x_t^\prime Q x_t + \delta V_{t+1}(x_{t+1})$$

Use the transition law of:

$$x_{t+1} = A_{t+1}x_t + B_{t+1}r_t$$

Such that the network first appears in period $t = 1$ and transforms $x_0$ to $x_1$. The network at time $t+1$ then transforms $x_t$ into $x_{t+1}$.

This gives:

$$V_t(x_t) = \max_{r_t} -x_t^\prime Q x_t - r_t^\prime R r_t + \delta V_{t+1}(A_{t+1}x_t + B_{t+1}r_t)$$

Use $V_t(x_t) = -x_t^\prime K_t x_t$ to expand this as:

$$-x_t^\prime K_t x_t = \max_{r_t} -x_t^\prime Q x_t - r_t^\prime R r_t + \delta (A_{t+1}x_t + B_{t+1}r_t)^\prime K_{t+1} (A_{t+1}x_t + B_{t+1}r_t)$$

The first-order condition with respect to $r_t$ gives the following:

$$-(B_{t+1}^\prime K_{t+1} B_{t+1} + \frac{R}{\delta})r_t = B_{t+1}^\prime K_{t+1} A_{t+1} x_t$$

and so we compute $r^*_t$ as $L_t x_t$ where $L_t$ is defined as:

$$L_t = -(B_{t+1}^\prime K_{t+1} B_{t+1} + \frac{R}{\delta})^{-1} B_{t+1}^\prime K_{t+1} A_{t+1}$$

In this sense the strategy $L_t$ is defined by the result $K_{t+1}$ and network $A_{t+1}$, $B_{t+1}$ of the next interval.

This gives rise to the following recursive definition for $K_t$:

$$K_t = Q + \delta A_{t+1}^\prime (K_{t+1} - K_{t+1} B_{t+1}(B_{t+1}^\prime K_{t+1} B_{t+1} + \frac{R}{\delta})^{-1} B_{t+1}^\prime K_{t+1})A_{t+1}$$

def solve_dynamic(order, x_0): 
    # order: a vector of length T containing the network matrices from period t = 1 to t = T
    
    R = 0
    delta = 0.9
    T = 15 # will zoom this in later in the notebook
    
    A_basis_1 = np.array([
        [0.6 * 0.99, 0.2*0.99, 0.2*0.99],
        [0.5, 0.5, 0.0],
        [0.2, 0.0, 0.8]
    ], ndmin = 2)
    
    A_basis_2 = np.array([
        [0.6, 0.2, 0.2],
        [0.5, 0.5, 0.0],
        [0.2*0.99, 0.0*0.99, 0.8*0.99]
    ], ndmin = 2)
    
    A = [A_basis_1, A_basis_2]
    
    B_basis_1 = np.array([
        [0.01],
        [0.0],
        [0.0]
    ], ndmin = 2)
    
    B_basis_2 = np.array([
        [0.0],
        [0.0],
        [0.01]
    ], ndmin = 2)
    
    B = [B_basis_1, B_basis_2]
    
    n = x_0.size
    Q = (1 / n) * np.identity(n)
    K_t = Q
    K_sequence = [Q]
    
    # iterate backwards to construct K_t from period T-1 to period 0
    # let K_T = Q
    # K_0 is not used in the computation of L_t as L_0 uses K_1 but I compute it anyway
    for t in range(T): # iterate over T periods from T-1 to 0
        K_t = delta * A[order[(T-t)-1]].T @ (K_t - K_t @ B[order[(T-t)-1]] @ np.linalg.inv(B[order[(T-t)-1]].T @ K_t @ B[order[(T-t)-1]] + R / delta) @ B[order[(T-t)-1]].T @ K_t) @ A[order[(T-t)-1]] + Q
        K_sequence.insert(0, K_t)

    x_t = x_0
    r_ts = []
    payoff = 0
    payoffs = []
    x_ts = [x_0]

    # iterate forwards to construct opinion vectors and payoffs from t = 1 to t = T
    # compute L_t from t = 0 to t = T-1
    for t in range(T):
        # here order[t] is the selected A or B for period t+1 since the order vector, at index zero, contains t = 1
        L_t = -(np.linalg.inv(B[order[t]].T @ K_sequence[t+1] @ B[order[t]] + R / delta) @ B[order[t]].T @ K_sequence[t] @ A[order[t]])
        r_t = L_t @ x_t
        r_ts.append(r_t.item())
        payoff += (-1 * delta**t * (x_t.T @ Q @ x_t)).item() + (-1 * delta**t * (R * r_t**2)).item()
        payoffs.append(payoff)
        # again, order[t] is the network matrix at time t+1
        x_t_new = A[order[t]] @ x_t + B[order[t]] @ r_t
        x_ts.append(x_t_new)
        x_t = x_t_new
        
    return payoff, payoffs, r_ts
